Thatâ€™s interesting! Do those groups have anything to do with whether someone stays or goes? We need some fancier plotting code to shade in the points by their value of retain.

y = work['retain']
labels = ['quit','stay']
colors = ['cornflowerblue', 'slategrey']
for yi in [0,1]:
    idx = y == yi
    plt.scatter(X_df.loc[idx,'years worked'], X_df.loc[idx,'commute'], color = colors[yi],
                label = labels[yi])
plt.xlabel('Years worked')
plt.ylabel('Commute')
plt.legend()
